# Practica Machine Learning Python

Repositorio con una libreta de Google Colab que contiene ejemplos basicos de Machine Learning y redes neuronales en Python.

## Contenido

- Compuerta XOR con una red neuronal MLP.
- Clasificacion supervisada con el dataset Iris usando KNN.
- Clustering no supervisado con KMeans.
- Backpropagation aplicado al problema XOR con TensorFlow/Keras.

## Tecnologias

- Python
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow/Keras
- Google Colab

## Uso

Abre `Libreta_Machine_Learning.ipynb` en Google Colab o Jupyter Notebook y ejecuta las celdas en orden.
